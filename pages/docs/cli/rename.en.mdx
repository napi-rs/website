# Rename

import { Green, Rust } from '../../../components/chalk'

Rename the **NAPI-RS** project

## Usage

```sh
# CLI
napi rename [--options]
```

```typescript
// Programmatically
import { NapiCli } from '@napi-rs/cli'

new NapiCli().rename({
  // options
})
```
