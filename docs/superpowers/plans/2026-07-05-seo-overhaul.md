# SEO Overhaul Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add the missing source-level SEO signals to the napi.rs Void build — canonical + hreflang, real meta-description fallback, JSON-LD, git-based sitemap lastmod + alternates, per-page OG images, and head polish.

**Architecture:** Derivable head signals (canonical, hreflang, JSON-LD, description fallback, og:image) are injected in the existing `middleware/01.head.ts` post-`next()` HTML rewrite, using small pure modules under `lib/seo/`. Which locales serve a given route (for hreflang) and blog dates (for JSON-LD) are read from a build-time-baked, committed `lib/i18n/route-map.gen.ts`, produced by a manually-run generator and consumed by both the middleware and the sitemap. Per-page OG PNGs are generated by a Vite build plugin into `dist/client/og/`.

**Tech Stack:** Void 0.10 (Vite 8 + React 19) on Cloudflare Workers; plain-ESM Node build scripts; Vitest via `vite-plus`; `satori` + `@resvg/resvg-js` for OG images.

## Global Constraints

- Base URL is `https://napi.rs`. Brand string is exactly `NAPI-RS`.
- Locales: `en` (DEFAULT — served at ROOT, hrefs UNPREFIXED), `cn` (→ `<html lang>` `zh-CN`), `pt-BR` (→ `pt-BR`). `htmlLang('en')='en'`, `htmlLang('cn')='zh-CN'`, `htmlLang('pt-BR')='pt-BR'` (`lib/docs/locale.ts:107-109`).
- **Canonical/JSON-LD/hreflang/OG URL convention = the REAL served URL:** en unprefixed, cn/pt-BR prefixed, no trailing slash except root (`https://napi.rs/`). This is NOT the legacy always-`/en/`-prefixed `canonicalUrl()` in `middleware/01.head.ts:107-112` — that function's output (`.../en/…`) is the mismatch we are fixing.
- **Edge-safety:** anything imported by `middleware/01.head.ts` runs on Cloudflare Workers. It MUST NOT import `node:fs`, `node:child_process`, or any filesystem/process API. `lib/seo/*` and `lib/i18n/route-map.gen.ts` must be pure data/string logic. `lib/docs/locale.ts` and `lib/nav/index.ts` are already edge-safe and importable.
- **Baked-module pattern** (`lib/i18n/route-map.gen.ts`): committed to git, generated by a manually-run script, NOT a Vite plugin (the middleware imports it into the worker bundle, so it must exist before bundling), and NOT regenerated at `void deploy`. Header banner: `// GENERATED by scripts/<x>.mjs — DO NOT EDIT.` + purpose + `// Regenerate with: node scripts/<x>.mjs`. Format the emitted file with `execSync('vp fmt "<file>" --write')` in a try/catch (mirror `scripts/build-nav.mjs:253-257`).
- **`void deploy` runs `vp build` (Vite), not `npm run build`** — only plugins registered in `vite.config.ts` run at deploy. OG PNGs (build OUTPUT under `dist/client/og/`) are therefore a Vite `writeBundle` plugin; the route map (committed SOURCE) is not.
- Middleware injection keeps its idempotency guard: skip re-injection when `html.includes('property="og:url"')` is already true (`middleware/01.head.ts:183`). Everything new is emitted in the same single injected block.
- **Tests:** Vitest via vite-plus, node env. Test files begin with `// @vitest-environment node` and import from `'vitest'` or `'vite-plus/test'`. Run a single file with: `GITHUB_TOKEN=dummy vp test run <path>`.
- **Formatting:** `vp fmt <file> --write` (oxfmt: no semicolons, single quotes, trailing comma `all`, printWidth 80).
- Do NOT alter the byte-0 `<script>` structure of the fmt-exempt island `.md` pages (`vite.config.ts:293-300`): `pages/en/docs/concepts/webassembly.md`, `pages/en/blog/announce-v2.md`, `pages/cn/blog/announce-v2.md`, `pages/en/blog/announce-v3.md`.
- OG images: `satori@^0.26` + `@resvg/resvg-js@^2.6`, canvas 1200×630, dark `#111111` background, accent `#e66000` (`hsl(26 100% 45%)`), white/`#fafafa` text. satori has NO system-font fallback — a `.ttf` font MUST be shipped in-repo.

```ts
// Pure, edge-safe SEO URL helpers. NO fs / node imports — imported by
// middleware/01.head.ts, which runs on Cloudflare Workers.
import { splitLocale, getLocale, DEFAULT_LOCALE } from '../docs/locale.ts'
import type { Locale } from '../nav/index.ts'

export const BASE_URL = 'https://napi.rs'

/**
 * Locale-neutral route key for a public path: locale prefix removed, leading
 * slash preserved, no trailing slash. Home and any `/en/…` normalise to the
 * unprefixed form. `/cn/docs/x` -> `/docs/x`, `/cn` -> `/`.
 */
export function neutralPath(publicPath: string): string {
  const [, rest] = splitLocale(publicPath)
  return rest ? `/${rest}` : '/'
}

/** Absolute URL of a neutral path in a given locale (en at root, others prefixed). */
export function localeUrl(neutral: string, locale: Locale): string {
  const suffix = neutral === '/' ? '' : neutral
  if (locale === DEFAULT_LOCALE) {
    return suffix ? `${BASE_URL}${suffix}` : `${BASE_URL}/`
  }
  return `${BASE_URL}/${locale}${suffix}`
}

/** The real served (canonical) URL for a public path. */
export function selfCanonical(publicPath: string): string {
  return localeUrl(neutralPath(publicPath), getLocale(publicPath))
}
```

```js
// Generates lib/i18n/route-map.gen.ts (committed, imported by the edge
// middleware + the sitemap). One-shot manual/CI generator like build-nav.mjs —
// NOT a Vite plugin and NOT run by `void deploy`. Regenerate after adding,
// removing, or translating any page (and after editing a blog `date:`).
//
// Regenerate with: node scripts/build-route-map.mjs
import { readFileSync, writeFileSync, existsSync } from 'node:fs'
import { join, dirname, resolve } from 'node:path'
import { fileURLToPath, pathToFileURL } from 'node:url'
import { execSync } from 'node:child_process'
import { walkPages, fileToRoute } from './generate-sitemap.mjs'
import { splitLocale } from '../lib/docs/locale.ts'

const __dirname = dirname(fileURLToPath(import.meta.url))
const root = resolve(__dirname, '..')
const pagesDir = join(root, 'pages')
const outFile = join(root, 'lib', 'i18n', 'route-map.gen.ts')

/** Group public routes by neutral path -> sorted unique locales. Pure. */
export function buildRouteMap(routes) {
  const acc = {}
  for (const route of routes) {
    const [locale, rest] = splitLocale(route)
    const neutral = rest ? `/${rest}` : '/'
    ;(acc[neutral] ??= new Set()).add(locale)
  }
  const out = {}
  for (const key of Object.keys(acc).sort()) out[key] = [...acc[key]].sort()
  return out
}

/** Pull a `date:` string from a blog `.md` (frontmatter may follow a <script>). Pure. */
export function parseBlogDate(src) {
  const m = src.match(
    /^\s*---[\s\S]*?\ndate:\s*['"]?([0-9]{4}-[0-9]{2}-[0-9]{2})['"]?/m,
  )
  return m ? m[1] : null
}

function generate() {
  const files = walkPages(pagesDir)
  const routes = []
  for (const rel of files) {
    const route = fileToRoute(rel)
    if (route !== null) routes.push(route)
  }
  const routeMap = buildRouteMap(routes)

  const blogDates = {}
  for (const rel of files) {
    if (!rel.endsWith('.md') || !rel.includes('/blog/')) continue
    const route = fileToRoute(rel)
    if (route === null) continue
    const [, rest] = splitLocale(route)
    const neutral = rest ? `/${rest}` : '/'
    const date = parseBlogDate(readFileSync(join(pagesDir, rel), 'utf8'))
    if (date) blogDates[neutral] = date
  }

  const ts = `// GENERATED by scripts/build-route-map.mjs — DO NOT EDIT.
// Which locales really serve each route (for hreflang + sitemap alternates) and
// blog publish dates (for BlogPosting JSON-LD). Data-only + edge-safe.
// Regenerate with: node scripts/build-route-map.mjs
import type { Locale } from '../nav/index.ts'

export const routeMap: Record<string, Locale[]> = ${JSON.stringify(routeMap, null, 2)}

export const blogDates: Record<string, string> = ${JSON.stringify(blogDates, null, 2)}
`
  writeFileSync(outFile, ts, 'utf8')
  try {
    execSync(`vp fmt "${outFile}" --write`, { cwd: root, stdio: 'pipe' })
  } catch {
    // vp not on PATH; pre-commit hook will format it.
  }
  console.log(
    `route-map: ${Object.keys(routeMap).length} routes, ${Object.keys(blogDates).length} blog dates -> ${outFile}`,
  )
}

const cliEntry = process.argv[1]
if (cliEntry && import.meta.url === pathToFileURL(cliEntry).href) generate()
```

```ts
// Pure, edge-safe hreflang builder. Reciprocal language alternates + x-default.
import { routeMap } from '../i18n/route-map.gen.ts'
import { localeUrl, neutralPath } from './urls.ts'
import { htmlLang } from '../docs/locale.ts'
import type { Locale } from '../nav/index.ts'

const ORDER: Locale[] = ['en', 'cn', 'pt-BR']

export interface HreflangAlt {
  hreflang: string
  href: string
}

/**
 * Ordered hreflang alternates for every locale that really serves the page,
 * plus `x-default` -> en. Empty when the route has < 2 locales. Single source
 * of truth — reused by hreflangLinks (head) and the sitemap generator.
 */
export function hreflangAlternates(
  publicPath: string,
  map: Record<string, Locale[]> = routeMap,
): HreflangAlt[] {
  const neutral = neutralPath(publicPath)
  const locales = map[neutral]
  if (!locales || locales.length < 2) return []
  const alts: HreflangAlt[] = ORDER.filter((l) => locales.includes(l)).map(
    (l) => ({ hreflang: htmlLang(l), href: localeUrl(neutral, l) }),
  )
  if (locales.includes('en')) {
    alts.push({ hreflang: 'x-default', href: localeUrl(neutral, 'en') })
  }
  return alts
}

/** The alternates rendered as `<link rel="alternate" hreflang>` tags. */
export function hreflangLinks(
  publicPath: string,
  map: Record<string, Locale[]> = routeMap,
): string {
  return hreflangAlternates(publicPath, map)
    .map(
      (a) => `<link rel="alternate" hreflang="${a.hreflang}" href="${a.href}">`,
    )
    .join('')
}
```

```ts
// Pure, edge-safe JSON-LD builder. Chooses schema by route shape.
import { selfCanonical, neutralPath, localeUrl, BASE_URL } from './urls.ts'
import { getLocale, htmlLang } from '../docs/locale.ts'
import { nav } from '../nav/index.ts'
import { getBreadcrumbCore } from '../docs/page-data.ts'
import { routeMap, blogDates } from '../i18n/route-map.gen.ts'

const ORGANIZATION = {
  '@type': 'Organization',
  name: 'NAPI-RS',
  url: BASE_URL,
  logo: `${BASE_URL}/img/favicon.png`,
  sameAs: [
    'https://github.com/napi-rs/napi-rs',
    'https://twitter.com/napi_rs',
    'https://discord.gg/SpWzYHsKHs',
  ],
}

const EXISTS = new Set(Object.keys(routeMap).map((p) => p.replace(/^\//, '')))

function breadcrumbList(publicPath: string) {
  const locale = getLocale(publicPath)
  const leaf = neutralPath(publicPath).replace(/^\//, '')
  const crumbs = getBreadcrumbCore(leaf, locale, nav[locale], (p) =>
    EXISTS.has(p),
  )
  if (!crumbs.length) return null
  return {
    '@type': 'BreadcrumbList',
    itemListElement: crumbs.map((c, i) => ({
      '@type': 'ListItem',
      position: i + 1,
      name: c.label,
      ...(c.href
        ? { item: c.href.startsWith('http') ? c.href : `${BASE_URL}${c.href}` }
        : {}),
    })),
  }
}

export function jsonLdFor(
  publicPath: string,
  { title, description }: { title: string; description: string },
): string {
  const url = selfCanonical(publicPath)
  const inLanguage = htmlLang(getLocale(publicPath))
  const neutral = neutralPath(publicPath)
  const graph: unknown[] = []

  if (neutral === '/') {
    graph.push(
      { '@type': 'WebSite', name: 'NAPI-RS', url, inLanguage, description },
      ORGANIZATION,
    )
  } else if (neutral.startsWith('/docs/')) {
    graph.push({
      '@type': 'TechArticle',
      headline: title,
      description,
      inLanguage,
      url,
      publisher: ORGANIZATION,
    })
    const bc = breadcrumbList(publicPath)
    if (bc) graph.push(bc)
  } else if (neutral.startsWith('/blog/')) {
    const date = blogDates[neutral]
    graph.push({
      '@type': 'BlogPosting',
      headline: title,
      description,
      inLanguage,
      url,
      author: { '@type': 'Organization', name: 'NAPI-RS' },
      publisher: ORGANIZATION,
      ...(date ? { datePublished: date } : {}),
    })
  } else {
    return '' // changelog + anything else: no JSON-LD
  }

  const ld = { '@context': 'https://schema.org', '@graph': graph }
  return `<script type="application/ld+json">${JSON.stringify(ld)}</script>`
}
```

```ts
// Pure orchestrator: the full <head> injection block for a page. Edge-safe.
import { selfCanonical, neutralPath, localeUrl } from './urls.ts'
import { hreflangLinks } from './hreflang.ts'
import { jsonLdFor } from './jsonld.ts'

export interface SeoHeadArgs {
  publicPath: string
  title: string
  description: string
  hasDescriptionMeta: boolean
  isFallback: boolean
  mdLink: string
}

export function buildSeoHead({
  publicPath,
  title,
  description,
  hasDescriptionMeta,
  isFallback,
  mdLink,
}: SeoHeadArgs): string {
  const canonical = isFallback
    ? localeUrl(neutralPath(publicPath), 'en')
    : selfCanonical(publicPath)
  const descMeta = hasDescriptionMeta
    ? ''
    : `<meta name="description" content="${description}">`
  return (
    descMeta +
    `<link rel="canonical" href="${canonical}">` +
    hreflangLinks(publicPath) +
    jsonLdFor(publicPath, { title, description }) +
    mdLink +
    `<meta property="og:title" content="${title}">` +
    `<meta property="og:description" content="${description}">` +
    `<meta property="og:url" content="${canonical}">`
  )
}
```

```js
// Build-time per-page OG image generator (satori -> resvg). Emits PNGs into
// dist/client/og/<route>.png as a Vite writeBundle plugin (mirrors sitemapPlugin).
// Docs + blog routes only; islands keep the static /img/og-v2.png.
import { readFileSync, writeFileSync, mkdirSync } from 'node:fs'
import { join, dirname, resolve, basename } from 'node:path'
import { fileURLToPath } from 'node:url'
import satori from 'satori'
import { Resvg } from '@resvg/resvg-js'
import React from 'react'
import { walkPages, fileToRoute } from './generate-sitemap.mjs'
import { splitLocale } from '../lib/docs/locale.ts'

const __dirname = dirname(fileURLToPath(import.meta.url))
const root = resolve(__dirname, '..')
const pagesDir = join(root, 'pages')

const MANROPE = readFileSync(join(root, 'assets/fonts/Manrope-Bold.ttf'))
const INTER = readFileSync(join(root, 'assets/fonts/Inter-SemiBold.ttf'))
const TITLE_SUFFIX_RE = /\s+[–—-]\s+NAPI-RS\s*$/

function ogTemplate(title) {
  const h = React.createElement
  return h(
    'div',
    {
      style: {
        width: '100%',
        height: '100%',
        display: 'flex',
        flexDirection: 'column',
        justifyContent: 'space-between',
        background: '#111111',
        padding: '80px',
        color: '#fafafa',
      },
    },
    h('div', {
      style: { width: '120px', height: '10px', background: '#e66000' },
    }),
    h(
      'div',
      {
        style: {
          fontFamily: 'Manrope',
          fontSize: 72,
          fontWeight: 700,
          display: 'flex',
        },
      },
      title,
    ),
    h(
      'div',
      {
        style: {
          fontFamily: 'Inter',
          fontSize: 32,
          color: '#e66000',
          display: 'flex',
        },
      },
      'napi.rs',
    ),
  )
}

/** Render a title card to a PNG Buffer. */
export async function renderOg(title) {
  const svg = await satori(ogTemplate(title), {
    width: 1200,
    height: 630,
    fonts: [
      { name: 'Manrope', data: MANROPE, weight: 700, style: 'normal' },
      { name: 'Inter', data: INTER, weight: 600, style: 'normal' },
    ],
  })
  return new Resvg(svg).render().asPng()
}

/** Public routes (docs+blog only) with their page title. */
function ogRoutes() {
  const out = []
  for (const rel of walkPages(pagesDir)) {
    if (!rel.endsWith('.md')) continue
    const route = fileToRoute(rel)
    if (route === null) continue
    const [, rest] = splitLocale(route)
    if (!(rest.startsWith('docs/') || rest.startsWith('blog/'))) continue
    const src = readFileSync(join(pagesDir, rel), 'utf8')
    const m = src.match(/^title:\s*['"]?(.+?)['"]?\s*$/m)
    const title = (m ? m[1] : rest).replace(TITLE_SUFFIX_RE, '')
    out.push({ route, title })
  }
  return out
}

export function ogImagePlugin() {
  return {
    name: 'napi-rs-og-images',
    apply: 'build',
    async writeBundle(options) {
      if (!options.dir || basename(options.dir) !== 'client') return
      let n = 0
      for (const { route, title } of ogRoutes()) {
        const png = await renderOg(title)
        const outPath = join(
          options.dir,
          'og',
          `${route.replace(/^\//, '')}.png`,
        )
        mkdirSync(dirname(outPath), { recursive: true })
        writeFileSync(outPath, png)
        n++
      }
      console.log(
        `napi-rs-og-images: ${n} images -> ${join(options.dir, 'og')}`,
      )
    },
  }
}
```

```ts
import { BASE_URL, neutralPath } from './urls.ts'

const FALLBACK = `${BASE_URL}/img/og-v2.png`

/** Per-page generated OG image URL for docs/blog, else the static fallback. */
export function ogImageUrl(publicPath: string): string {
  const neutral = neutralPath(publicPath)
  if (!(neutral.startsWith('/docs/') || neutral.startsWith('/blog/'))) {
    return FALLBACK
  }
  const served = publicPath.replace(/\/+$/, '').replace(/^\//, '')
  return `${BASE_URL}/og/${served}.png`
}
```

Note: `publicPath` is the served path (en unprefixed, cn/pt-BR prefixed), which matches the generator's output route, so `served` maps 1:1 to the emitted PNG.
